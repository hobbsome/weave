WVRequest

public class WVRequest

The basic request class. Doesn’t have any special bells or whistles. Can parse JSON and Strings.

    Parameters

    url

    After the request finishes, this gets called.
    requestType

    The request type to make. Defaults to GET
    outputType

    The request output type. Defaults to string.
    timeoutInterval

    How long the request tries for. Defaults to 10 seconds.
    parameters

    Any HTTP body to send. Defaults to bank.
    headers

    Any HTTP headers to send. Defaults to blank.
    finishHandler

    After the request finishes, this gets called. Provides a WVResponse. You can cast it to the request type you requested in requestType.
