What is Weave?

Weave is a open-source networking layer build on top of NSURLSession. It is lightweight yet still has useful features. When designing Weave, I had three main goals:

  1. Provide powerful features that weren’t in any other framework.
  2. Provide simplicity in implementation and API.
  3. Make it easy for users to integrate and use. ## Documentation Documentation is available at hobbsome.com/weave. ## Get Started & Installation There are two main different ways to use Weave:
  4. Build & maintain from source
  5. Download framework ### Build and Integrate
  6. To build, clone the entire project.
  7. Make sure you’re working in a Xcode Workspace.
  8. Then, simply drag the .xcodeproj onto the File Browser in Xcode.
  9. Add Weave to the Embedded Frameworks section of your project’s .xcodeproj.
  10. Have fun! ### Download
  11. Download the latest release.
  12. Drag it into your .xcodeproj.
  13. Add it to your Embedded Frameworks section in your project’s .xcodeproj.
  14. Have fun!
